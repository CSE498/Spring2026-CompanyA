## These listed files (BSP-Dungeon-Seed-xxxxx.md) are meant to provide quick info on the BSP-Tree's information (x+y coordinates, width/height, room information)
## as well as the locations of the rooms in the grid for testing purposes

## Having the information in this directory is a lot more accessible instead of having to change the BSP-Tree's seed on every re-compile...

# General Info

## The number of Leaf nodes determine the number of rooms in the BSPTree
## The total number of Nodes within the BSP-Tree determine the number of splits that've occurred in the grid
