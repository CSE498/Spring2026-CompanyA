# Instructions for SDL download - NECESSARY

This download is crucial for compiling and running our group's files/tests. Please follow these steps:

1) Open IDE terminal
2) Navigate to top-level directory (Spring2026-CompanyA)
3) Run the command 'make install' (this will take a minute to import & create all necessary files)

Important side note: 'make install-clean' will remove the third-party/SDL2 & third-party/sdl-build directories & 
all their subdirectories. Only run this if updating the SDL version or if something breaks during install.

Steps to run the tests:

1) Open IDE terminal
2) Navigate to the /tests directory
3) Run the command 'make'
